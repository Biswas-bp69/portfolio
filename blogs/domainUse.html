How to Active and use domain

Below is how to use Github and put a project on Github. |

If the domain name has been received and it needs to be kept in the repository:

1. github page enable गर्ने:

  • Open your github Repo
  • Go To setting and page
  • Under score:
    • branch: main
    • Folder: /root
  • Click Save

    • Note: GitHub will generate a template link : "https://username.github.io/repo-name"

2.Add Custom Domain in GitHub

  • Go to github repo
  • Go to setting => Page
  • In Custom Domain: yourDomain.com.np
  • Click Save

    • Note: It May show an error until DNS is configured this is normal.

3. Setup DNS in Claudfare(Most Imp)

  • Open Claudfare Dashboard(you do login)
  • Select Your Domain
  • Go to DNS => Record
  • Add the Following Records:

  • For CNAME Record
    • type: CNAME
    • Name: www
    • Target: username.github.io
    • Proxy Status: DNS only (Gray cloud)
  • A Record (for root domain)
    • Type: A
    • Name: @
    • TPv4: 185.199.108.153
    • proxy: DNS Only
  • A Record (for root domain)
    • Type: A
    • Name: @
    • TPv4: 185.199.109.153
    • proxy: DNS Only
  • A Record (for root domain)
    • Type: A
    • Name: @
    • TPv4: 185.199.110.153
    • proxy: DNS Only
  • A Record (for root domain)
    • Type: A
    • Name: @
    • TPv4: 185.199.111.153
    • proxy: DNS Only

4. Create CNAME file in your repo

  • Click Add file => Create new file
  • File name: CNAME
  • File content your domain ( eg: example.com.np )
  • Commit the file
