How to use GitHub

1. Go to https://github.com and create an account.
2.Download git from https://git-scm.com.
3. Create a repository on Github
  • Open github
  • Click top right side "+" icons
  • Click new repository
  • Type repo name
  • Choose private or public
  • Click creat repo
4. Two ways to upload a project
First method (easy way)
  • Open repo
  • Click on upload file
  • Choose your file
  • Click commit
Second methord (Professional way)
  • Open vs code
  • Open project
  • Go to command prompt ( "ctrl" + "~")
  • Command: git branch branch-name
  • Command: git branch ( check all branch)
  • Command: git checkout branch-name
  • Command: git init
  • Commmand: git add .
  • Commmand: git commit -m "something"
  • Commmand: git push -u origin branch-name
